# Simple Forum

## Simple forum app built using React, Express and MongoDB.

### Features
User Authentication 
Create new posts
View posts
View profiles
View specific user posts

### Run locally
    git clone https://github.com/nbapera/forum/
    cd forum
    npm install & npm start
    cd ../backend
    npm install & npm start
